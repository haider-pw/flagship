Console Chat

Easy chat for your web site

For install you need:

  • Copy files in your project
  • Add two includes in you site header, after jquery
    <script type="text/javascript" src="/console_chat/js/scripts.js"></script>
    <link href="/console_chat/css/styles.css" rel="stylesheet">
  • Make folders "cache" and "history" writable
  • Configurate options on chat.php (Admin password, history limit and set, using unique chat for all pages or not)
  • Configurate js in /js/scripts.js (Keys for open\close chat, default username, link for request new sessages and others)
  • Add in html:
    <script>
    $(document).ready(function(){
    consoleChat('username');// username not necessarily, default name Guest
    });
    </script>
  • Open you web page and click Ctrl+` (or other keys, if you change) or click chat button ot top, if then enabled
  • Enjoy

Commands

/help - watch this text
/name - change name
/admin - try to login as admin
/ban - ban
/banip - ban ip
/unban - unabn ip or name
/unbanall - Unban all names and ips
/bannednames - show all banned names
/bannedips - show all banned ips
/w %username% %message% - send private message
For login as admin, try msg "/admin 123" (123 is default password, you can change it on chat.php)
If you logined as admin, you can click on other usernames and ban it.
Use command /unban %username% or /unbanall for set it free.
